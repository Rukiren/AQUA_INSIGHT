How to Use This?

1. go to import_file/command.py
2. find and change to your openai api key
```
# 設定 OpenAI API 的金鑰
OPENAI_API_KEY = "//Your open api key//"
```
3. back to / and install requirement
4. python3 app.py
5. open browser and connect URL
6. try it!
